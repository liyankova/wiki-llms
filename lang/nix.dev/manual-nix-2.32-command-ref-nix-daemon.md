# nix-daemon - Nix 2.32.2 Reference Manual

# [Name](https://nix.dev/manual/nix/2.32/command-ref/nix-daemon#name)

`nix-daemon` - Nix multi-user support daemon

# [Synopsis](https://nix.dev/manual/nix/2.32/command-ref/nix-daemon#synopsis)

`nix-daemon`

# [Description](https://nix.dev/manual/nix/2.32/command-ref/nix-daemon#description)

The Nix daemon is necessary in multi-user Nix installations. It runs
build tasks and other operations on the Nix store on behalf of
unprivileged users.
