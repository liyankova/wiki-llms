# C API - Nix 2.32.2 Reference Manual

# [C API](https://nix.dev/manual/nix/2.32/c-api#c-api)

Nix provides a C API with the intent of [*becoming*](https://github.com/NixOS/nix/milestone/52) a stable API, which it is currently not.
It is in development.
