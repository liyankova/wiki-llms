# Supported Platforms - Nix 2.32.2 Reference Manual

# [Supported Platforms](https://nix.dev/manual/nix/2.32/installation/supported-platforms#supported-platforms)

Nix is currently supported on the following platforms:

* Linux (i686, x86\_64, aarch64).
* macOS (x86\_64, aarch64).
