# Glossary

# Glossary[#](https://nix.dev/reference/glossary#glossary "Link to this heading")

Nix[#](https://nix.dev/reference/glossary#term-Nix "Link to this term")
:   Build system and package manager.

    Read /nɪks/ (“Niks”).

    See also

    * [Nix reference manual](https://nix.dev/reference/nix-manual)
    * [Nix source code](https://github.com/NixOS/nix)

Nix language[#](https://nix.dev/reference/glossary#term-Nix-language "Link to this term")
:   Programming language to declare packages and configurations for [Nix](https://nix.dev/reference/glossary#term-Nix).

    See also

    * [Nix language basics](https://nix.dev/tutorials/nix-language#reading-nix-language)
    * [Nix language reference](https://nix.dev/manual/nix/stable/language)

Nix expression[#](https://nix.dev/reference/glossary#term-Nix-expression "Link to this term")
:   Expression written in the [Nix language](https://nix.dev/reference/glossary#term-Nix-language).

Nix file[#](https://nix.dev/reference/glossary#term-Nix-file "Link to this term")
:   File (`.nix`) containing a [Nix expression](https://nix.dev/reference/glossary#term-Nix-expression).

Nixpkgs[#](https://nix.dev/reference/glossary#term-Nixpkgs "Link to this term")
:   Software distribution built with [Nix](https://nix.dev/reference/glossary#term-Nix).

    Read /nɪks ˈpækɪʤɪz/ (“Nix packages”).

    See also

    * [Nixpkgs reference manual](https://nixos.org/manual/nixpkgs)
    * [Nixpkgs source code](https://github.com/NixOS/nixpkgs)

NixOS[#](https://nix.dev/reference/glossary#term-NixOS "Link to this term")
:   Linux distribution based on [Nix](https://nix.dev/reference/glossary#term-Nix) and [Nixpkgs](https://nix.dev/reference/glossary#term-Nixpkgs).

    Read /nɪks oʊ ɛs/ (“Niks Oh Es”).

    See also

    * [NixOS reference manual](https://nixos.org/manual/nixos)
    * [NixOS source code](https://github.com/NixOS/nixpkgs/tree/master/nixos)
