# Nix reference manual[#](https://nix.dev/reference/nix-manual#nix-reference-manual "Link to this heading")

The Nix reference manual is available for multiple versions:

* [Nix pre-release](https://nix.dev/manual/nix/development/)

  Development build from the `master` branch of the [Nix repository](https://github.com/NixOS/nix)
* [Nix 2.32](https://nix.dev/manual/nix/latest/) ([single page](https://nix.dev/manual/nix/latest/nix-2.32.html))

  Latest Nix release
* [Nix 2.31](https://nix.dev/manual/nix/rolling/) ([single page](https://nix.dev/manual/nix/rolling/nix-2.31.html))

  Shipped with the rolling release of [Nixpkgs](https://nix.dev/reference/glossary#term-Nixpkgs) and [NixOS](https://nix.dev/reference/glossary#term-NixOS)
* [Nix 2.28](https://nix.dev/manual/nix/stable/) ([single page](https://nix.dev/manual/nix/stable/nix-2.28.html))

  Shipped with the current stable release of [Nixpkgs](https://nix.dev/reference/glossary#term-Nixpkgs) and [NixOS](https://nix.dev/reference/glossary#term-NixOS): 25.05
* [Nix 2.24](https://nix.dev/manual/nix/prev-stable/) ([single page](https://nix.dev/manual/nix/prev-stable/nix-2.24.html))

  Shipped with the previous stable release of [Nixpkgs](https://nix.dev/reference/glossary#term-Nixpkgs) and [NixOS](https://nix.dev/reference/glossary#term-NixOS): 24.11
