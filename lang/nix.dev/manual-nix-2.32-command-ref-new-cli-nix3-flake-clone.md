# nix flake clone - Nix 2.32.2 Reference Manual

> **Warning**   
> This program is
> [**experimental**](https://nix.dev/manual/nix/2.32/development/experimental-features#xp-feature-nix-command)
> and its interface is subject to change.

# [Name](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#name)

`nix flake clone` - clone flake repository

# [Synopsis](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#synopsis)

`nix flake clone` [*option*...] *flake-url*

# [Examples](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#examples)

* Check out the source code of the `dwarffs` flake and build it:

  ```
  # nix flake clone dwarffs --dest dwarffs
  # cd dwarffs
  # nix build
  ```

# [Description](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#description)

This command performs a Git or Mercurial clone of the repository
containing the source code of the flake *flake-url*.

# [Options](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#options)

* [`--dest`](https://nix.dev/manual/nix/2.32/command-ref/new-cli/nix3-flake-clone#opt-dest) / `-f` *path*

  Clone the flake to path *dest*.
